# mongoose-crud

### list of books routes :

ROUTE           | HTTP      |  Description
--------------- | --------- | -------------         
/api/books      |  GET      | Get All Books
/api/books      |  GET      | Get One Books
/api/books      |  POST     | Create a Books
/api/books/:id  |  DELETE   | Delete a Books
/api/books/:id  |  PUT      | Update a Books
