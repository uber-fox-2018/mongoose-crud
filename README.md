# mongoose-crud

## Books
### List of routes

Route | HTTP | Description 
----- | ---- | -----------
/books | GET | Get all the book's data
/books | POST | Create a Book
/books/:id | PUT | Update the book
/books/:id | DELETE | Delete the book
